# Selfdev-Avatar

A talking face generation for SelfDev platform

This project is based on [SadTalker](https://github.com/OpenTalker/SadTalker/). SadTalker is a learning Realistic 3D Motion Coefficients for Stylized Audio-Driven Single Image Talking Face Animation. There is an [SadTalker App](https://huggingface.co/spaces/vinthony/SadTalker) online.
